 ### COMANDOS GIT ###

### 1. Git init
   Cria um novo repositório git na pasta atual
   Ajuda: git help init

### 2. Git status
   mostra o atual estado do repositório
   Ajuda: git help status

### 3. Git add
   adiciona arquivos e prepara para commitar
   Ajuda: git help add

### 4. Git commit -m
   Registra as alterações no histórico com uma mensagem  
   Ajuda: git help commit -m

### 5. Git log
   mostra o histórico dos commits
   Ajuda: git help log

### 6. Git config
   usado para configurar o git
   Ajuda: git help config

### 7. Git push
   Envia commits para o repositório remoto
   Ajuda: git help push

### 8. Git pull
   puxa mudanças do repositório remoto e mistura com o local
   Ajuda: git help pull

### 9: Git clone
   Clona um repositório remoto para sua máquina
   Ajuda: git help clone

### 10: Git bracnh
    Mostra ou cria branchs
    Ajuda: git helpe branch
