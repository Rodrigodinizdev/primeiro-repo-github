As respostas coletadas neste repositório são referentes aos estudos de:

- **Git/GitHub**: Versionamento de código, controle de versão, comandos Git, fluxos de trabalho colaborativo e boas práticas de uso do GitHub.
- **C# Básico**: Fundamentos da linguagem C#, sintaxe básica, tipos de dados, estruturas de controle e conceitos introdutórios de programação.
